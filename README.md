# LedgeStay

A rental listing platform helping users find, list, and manage accommodation stays with integrated enquiry management, admin approval workflow, and secure authentication.

# 🚀 LedgeStay

**Find and list rental stays near you — fast, simple, and verified.**

LedgeStay is a modern rental platform built to help users discover nearby stays and allow owners to list properties easily with real-time enquiries and admin verification.

---

## ✨ Features

* 🔐 Secure Authentication (OTP + JWT)
* 🏠 List & browse rental stays
* 📍 Location-based search (nearby stays)
* ❤️ Wishlist (save favorites)
* ⭐ Reviews & ratings
* 📩 Enquiry system (direct contact)
* 🛠 Admin approval workflow
* 🔔 Real-time notifications

---

## 🛠 Tech Stack

* **Frontend**: Vanilla JavaScript (SPA)
* **Backend**: Node.js + Express.js
* **Database**: MongoDB (Atlas)
* **Authentication**: JWT + Twilio OTP
* **Image Storage**: Cloudinary
* **Email Service**: Resend
* **Deployment**: Railway

---

## 🚀 Run Locally

```bash
git clone <your-repo-url>
cd ledge-stay
npm install
npm start
```

Server runs on:
👉 http://localhost:3000

---

## ⚙️ Environment Setup

Create a `.env` file in root:

```env
NODE_ENV=production
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret_key

CLOUDINARY_CLOUD_NAME=your_name
CLOUDINARY_API_KEY=your_key
CLOUDINARY_API_SECRET=your_secret

TWILIO_ACCOUNT_SID=your_sid
TWILIO_AUTH_TOKEN=your_token
TWILIO_VERIFY_SID=your_verify_sid

RESEND_API_KEY=your_key
EMAIL_FROM=your_email

APP_BASE_URL=http://localhost:3000
```

⚠️ Never upload `.env` to GitHub
